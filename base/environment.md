# 第一章   GitBook 基础环境

***

前面提及到 GitBook 是 Git  和 Markdown 结合产生的强大工具，那么需要安装以下环境：
* 安装 **Node.js**&emsp;（GitBook 是基于Node.js的）
* 安装 **Git**&emsp;（强大的版本控制软件）
* 安装 **GitBook**&emsp;（主角）
* 安装 **GitBook Editor**&emsp;（GitBook 的编辑器）

所有的这些环境可以在一台主机上完成，我们尽量按照最简单的方式安装和测试。
